# Product Overview

TournaGen is a static tournament generator web application that helps organizers create, visualize, and manage various tournament formats entirely in the browser. The application targets hobby sports leagues, esports communities, school tournaments, and online events.

## Key Features

- Multiple tournament formats: Single/Double Elimination, Round Robin, Swiss, Free-for-All, FIFA-style, Racing (Mario Kart & F1)
- Client-side only architecture (no backend required)
- Import/export tournament configurations as JSON
- Real-time visualization with connecting lines showing progression
- Responsive design for desktop, tablet, and mobile
- Offline functionality after initial load
- Local storage persistence

## Core Principles

- Modularity first: Tournament formats as plugins
- Static by default with targeted interactivity
- Fine-grained reactivity for performance
- Zero backend dependency
- Visual clarity through consistent design
